Java8 Try

To wrap checked exceptions and to make failures explicit. This is a translation of Scala's Try[T]. Basic api to do with container semantics like transformation, filtering, flatMap and failure recovery are present as of date, while other is being evolved.

Build Info

Use Gradle 2.0 for builds. Please do not checkin Eclipse or Intellij or any IDE specific files.
For Idea or Eclipse they can be generated using

  • gradlew eclipse
  • gradlew idea

Project Versioning

We will be following JBoss Versioning Convention

  • major.minor.micro.Alpha[n]
  • major.minor.micro.Beta[n]
  • major.minor.micro.CR[n] Please refer to AppConfig.groovy - a single place of change for all the project configuration changes
